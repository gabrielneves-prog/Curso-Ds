<!--

=========================================================

0. BARRA DE NAVEGAÇÃO (MENU SUPERIOR)

=========================================================

Para que serve esta seção?

É uma barra fixa no topo da página, com uma borda inferior

separando-a do restante do conteúdo. Ela funciona como um

"menu de opções do perfil": links rápidos que levam o

visitante direto para cada seção do currículo, sem precisar

rolar a página manualmente.

Por que isso deixa o site mais profissional?

Sites de portfólio e currículos online costumam ter um menu

fixo no topo. Isso facilita a navegação do recrutador e

mostra organização por parte de quem criou o site.

Como funciona o link (href="#id")?

Cada link abaixo aponta para o "id" de uma seção já existente

no HTML (ex: id="sobre", id="tecnologias"). Ao clicar, o

navegador rola automaticamente até aquele trecho da página.

Não é necessário JavaScript para isso, é um recurso nativo do HTML.

O QUE ALTERAR AQUI:

- "Seu Nome" (logo em texto) -> pode trocar pelo seu nome

ou apenas suas iniciais (ex: "S.N.");

- Os textos dos links (Sobre, Tecnologias, etc.) -> mantenha

os mesmos "href", apenas troque o texto se quiser;

- Para ADICIONAR um novo botão, copie uma tag <a> completa

e cole antes do fechamento de </nav>.

- Para REMOVER um botão, apague a tag <a> correspondente.

-->

<nav class="navbar-superior">

<div class="container navbar-conteudo">

<!-- Logo / nome em texto, do lado esquerdo da barra -->

<a href="#" class="navbar-logo"> Gabriel Ribeiro Neves </a>

=========================================================

=========================================================

Objetivo desta seção:

POR QUE AS DUAS FICAM JUNTAS NESTA SEÇÃO, MAS SEPARADAS

VISUALMENTE?

Hard Skills e Soft Skills são tipos diferentes de habilidade

(técnica x comportamental), mas fazem parte do mesmo bloco

do currículo. Por isso, usamos:

- UM único "painel-secao" (a caixa com borda) envolvendo

- DOIS subtítulos internos ("Hard Skills" e "Soft Skills"),

O QUE É O "BOTÃO DE STATUS" EMBAIXO DE CADA CARD?

Cada card de habilidade agora tem, embaixo do nome, um

pequeno botão indicando o nível de domínio:

- "Plena" -> classe btn-dourado (preenchido)

- "Em Desenvolvimento" -> classe btn-contorno (somente borda)

São as MESMAS classes de botão já usadas na seção de Projetos

(.btn-personalizado .btn-pequeno), então o visual fica idêntico.

A classe extra ".status-skill" apenas remove o efeito de "link

clicável" (cursor de mãozinha), já que aqui é só um indicador,

não uma ação.

Como ADICIONAR uma nova habilidade:

1. Copie um bloco completo <div class="col-6 col-md-3 mb-4"> ... </div>

3. Altere o texto dentro do <p> e, se necessário, o texto/

classe do botão de status.

Como REMOVER uma habilidade:

1. Apague o bloco <div class="col-6 col-md-3 mb-4"> ... </div>

inteiro correspondente.

Como ALTERAR o status de "Plena" para "Em Desenvolvimento" (ou vice-versa):

1. Troque o texto do botão (ex: "Plena" -> "Em Desenvolvimento");

contrário), mantendo sempre "btn-personalizado btn-pequeno status-skill".
